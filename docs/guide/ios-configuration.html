iOS configuration

NFC polling identifiers belong to the host app, not the Swift package. Keep the list scoped to the card families your app supports.

Required

  • Enable NFC Tag Reading.
  • Add NFCReaderUsageDescription.
  • Add com.apple.developer.nfc.readersession.formats with TAG.
  • Add FeliCa system codes and ISO 7816 AIDs as needed.
